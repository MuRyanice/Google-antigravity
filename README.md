## ğŸš€ Quick Start

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/MuRyanice/Google-antigravity.git
   cd Google-antigravity
   ```

2. **Start a local server**
   ```bash
   # Using Python 3
   python3 -m http.server 8080
   
   # Or using Node.js
   npx http-server -p 8080
   ```

3. **Open in browser**
   ```
   http://localhost:8080
   ```

### Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/MuRyanice/Google-antigravity)

Or manually:

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel --prod
```

## ğŸ¨ Customization

### Update Video

Replace the YouTube video ID in `index.html`:

```html
<iframe 
    src="https://www.youtube.com/embed/YOUR_VIDEO_ID"
    ...
</iframe>
```

Or add a local video file to `/videos/demo.mp4` (see `/videos/README.md` for details).

### Update Branding

1. **Logo**: Update the SVG in the nav and footer sections
2. **Colors**: Modify CSS variables in `:root` in `styles.css`
3. **Content**: Edit text directly in `index.html`
